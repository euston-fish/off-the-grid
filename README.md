# Off The Grid

> a work-in-progress entry for JS13kGames 2018

## Prerequisites

Ensure you have `make`, `npm`, `unzip`, [`google-closure-compiler`](https://www.npmjs.com/package/google-closure-compiler) and [`eslint`](https://eslint.org).

## Build

```
make
```

## Run

```
make run
```

## Develop

Edit files under `src`.
